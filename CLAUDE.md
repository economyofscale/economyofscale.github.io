# Daniel Fixemer — Resume Homepage: Build Brief

## Overview
Personal resume/portfolio homepage. Single-page, responsive (desktop + iOS/Android web).
Hosted free on GitHub Pages. Static HTML/CSS/JS (no backend needed).

## Design direction
- **Palette:** Charcoal / off-white base, with one accent color (to be chosen during build —
  suggest 2-3 options if not already decided, e.g. a muted blue, warm amber, or forest green).
- **Tone reference:** Structure inspired by maddieharris.org (clear sections, skills as grouped
  tag/pill clusters, smooth-scroll nav). Visual restraint and typography inspired by
  willwetherilt.com (confident headline, muted neutral palette, editorial feel, whitespace-led).
- Avoid: beige/warm-pastel tones, overly decorative UI chrome.
- Mobile-first responsive layout; works identically well on desktop browsers and iOS/Android
  mobile browsers (no native app, just responsive CSS).
- Some interactive/animated touches are welcome (smooth scroll, subtle hover/reveal animations,
  filterable elements) but should stay understated given the editorial tone.

**State of Geneva**

*Challenge:* The State of Geneva was replacing fragmented, manual planning processes with
OneStream as its strategic planning and budgeting platform — requiring deep adaptation to
public-sector needs: complex planning structures, workflow governance, and strict security
requirements.

*Approach:* Designed and built custom business rules and workflow automation for budgeting and

*Outcome:* A highly customized OneStream application meeting the canton's governance and security
requirements, replacing manual work with automation and controlled workflows, and giving business
users a centralized planning platform — delivered and expanded across multiple implementation
phases.

*Outcome:* A centralized planning and reporting platform that cut manual effort, improved data
consistency, and gave finance greater transparency into budgeting and forecasting — with a
